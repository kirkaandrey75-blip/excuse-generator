# excuse-generator
Генератор оправданий для работы — простой онлайн-инструмент, который помогает быстро придумать подходящее оправдание для начальника или коллег. Один клик — и готовое объяснение без регистрации и оплаты. Подходит для рабочих чатов, сообщений и неловких ситуаций, когда фантазия закончилась.
